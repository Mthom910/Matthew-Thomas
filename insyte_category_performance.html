What's Driving the AOV Change? — Prior 12mo → Trailing 12mo Bridge

Mix shift = effect of jobs shifting between categories with different typical order values, holding each category's own AOV fixed at the prior period. Price/discount = the remaining change, i.e. each category's own AOV moving.

What's Driving the Conversion Rate Change? — Prior 12mo → Trailing 12mo Bridge

Mix shift = effect of pipeline volume shifting toward categories with structurally different win rates, holding each category's own conversion rate fixed at the prior period. Win-rate effect = the remaining change, i.e. each category converting better or worse than before.
